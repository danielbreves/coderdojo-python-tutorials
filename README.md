# CoderDojo Python Tutorials 🐍

Learn Python by building games, one level at a time!

## Getting started

1. Click the green **Code** button at the top right of the page, then **Codespaces** > **Create codespace on main**
2. Wait for the codespace to load (this can take a couple of minutes) — it will open a code editor in your browser
3. On the left side you'll see a list of folders — start with **rock-paper-scissors**

## Tutorials

| Level | Tutorial | What you'll learn |
|-------|----------|-------------------|
| Beginner | [Rock, Paper, Scissors](rock-paper-scissors/level-1.py) | Variables, input, lists, loops, functions |
| Intermediate | [Battleship](battleship/level-1.py) | Classes, objects, methods, dictionaries |

Start with Rock, Paper, Scissors — Battleship builds on what you learn there!

## Editor setup

This project includes VSCode settings (in `.vscode/settings.json`) designed to keep things simple for beginners. The editor uses a Scratch-inspired colour theme so the syntax highlighting feels familiar to kids coming from Scratch, and the file explorer is decluttered by hiding dotfiles and other non-tutorial files. These settings apply automatically in VSCode and Codespaces.
