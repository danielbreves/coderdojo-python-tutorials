# Battleship 🚢

Learn about classes and objects by building a Battleship game!

This tutorial builds on what you learned in Rock, Paper, Scissors. Make sure you've finished that tutorial first.

## How it works

Each level teaches you something new. Read the instructions at the top of the file, write the code where it tells you, then run it to see what happens.

To run a level, type this in the **terminal** (the area at the bottom of the screen) and hit Enter:

```
python battleship/level-1.py
```

Replace the number with whichever level you're on.

## Levels

| Level | What you'll learn |
|-------|-------------------|
| 1 | Creating a class with attributes |
| 2 | Using `__init__` to set default values |
| 3 | Adding methods to a class |
| 4 | Creating a second class (Player) |
| 5 | Objects working together, dictionaries |
| 6 | Building the Game class, ship placement |
| 7 | The play loop with turn switching |
| 8 | Win condition and finishing the game |
| 9 | Bonus: ideas to try on your own! |
